This plugin provides information about users with a configurable role(by default the trainer) in a block in the course.

Installation:
To install the plugin just copy the folder "eledia_trainerinfo" into moodle/blocks/.

Afterwards you have to go to http://your-moodle/admin (Site administration -> Notifications) to trigger the installation process.

Usage:
In "Settings/Plugins/Blocks/Referee of this Course" you can set the role which will be displayed by this block.
Add the block to the course you want it to appear in.
